¡Bienvenido a **CocktailHub**! Este es un proyecto interactivo desarrollado como parte del módulo de **Lenguajes de Marcas** en el primer año del CFGS de Desarrollo de Aplicaciones Multiplataforma (DAM). 

La web ofrece una experiencia inmersiva para amantes de la coctelería, combinando una estética *Dark Mode* con efectos visuales avanzados.

---

## ✨ Características Principales

* **🃏 Sistema de Cartas Flip 3D:** Implementación de transformaciones en el eje Y para mostrar ingredientes al interactuar.
* **🎨 Diseño Premium Dark:** Inspirado en interfaces modernas como Eneba, utilizando variables CSS (`:root`) para una gestión de color eficiente.
* **🔍 Buscador Integrado:** Header funcional con barra de búsqueda y navegación optimizada.
* **📱 Totalmente Responsive:** Adaptabilidad garantizada mediante el uso de **Flexbox** y unidades relativas (`rem`, `vh`).
* **🏷️ Categorización por Tags:** Identificación visual rápida mediante etiquetas de estilo (Refrescante, Tropical, etc.).

## 🛠️ Tecnologías y Conceptos Aplicados

* **HTML5 Semántico:** Uso correcto de etiquetas estructurales para mejorar el SEO y la accesibilidad.
* **CSS Avanzado:** * **Flexbox:** Para el alineamiento del Header y la Grid de cócteles.
    * **Posicionamiento:** Uso de `relative` y `absolute` para capas y overlays.
    * **Efectos 3D:** Propiedades `perspective` y `backface-visibility`.
* **Arquitectura de Datos:** Organización limpia de carpetas para escalabilidad.
